# ロール：エンジニア
ヴェルナーガーデンに存在する「未知のウイルス」に感染し、奇跡的に適応した存在をシンバイオントと呼びます。
<br>
<br>
<br>

## ロール特技
エンジニアは主にセントリーガンを用いて戦闘を行います。
<br>
<br>

| 特技名 | 種類 | 消費AP | 内容 |
| :--- | :--- | :--- | :--- |
| セントリーガン | パッシブ | 0 | 戦闘中に1台、セントリーガン（AP2・攻撃力3・HP10）を設置することが可能。 |
| 銃身改造 | パッシブ | 0 | セントリーガンの攻撃力が2倍になる。 |
| 装甲強化 | パッシブ | 0 | セントリーガンのHP+5 |
| 軽量化 | パッシブ | 0 | セントリーガンのHPが5になるが、二つ設置できるようになる。 |

*社会的希少性ウイルス感染からシンバイオントとして生き残るケースは極めてレアであり、世界に数えられるほどしか存在しません。ラプラススタンダードがヴェルナーガーデンを忌避・警戒する最大の理由は、この「制御不能な異能者」を生み出す環境にあります。*
