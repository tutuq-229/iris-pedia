# ロール：エンジニア
ヴェルナーガーデンに存在する「未知のウイルス」に感染し、奇跡的に適応した存在をシンバイオントと呼びます。
<br>
<br>
<br>

## ロール特技
エンジニアは主にセントリーガンを用いて戦闘を行います。
<br>
<br>

| 特技名 | 種類 | 消費AP | 内容 |
| :--- | :--- | :--- | :--- |
| セントリーガン | 持続 | 0 | 戦闘中に1台まで、セントリーガン（AP2・攻撃力3・HP10）を設置することが可能。設置に3AP必要。 |
| 銃身改造 | 持続 | 0 | セントリーガンの攻撃力が2倍になる。 |
| 装甲強化 | 持続 | 0 | セントリーガンのHP+5 |
| 軽量化 | 持続 | 0 | セントリーガンのHPが5になるが、二つ設置できるようになる。 |
