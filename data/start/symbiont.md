# 🧬 シンバイオント (Symbiont)
ヴェルナーガーデンに存在する「未知のウイルス」に感染し、奇跡的に適応した存在をシンバイオントと呼びます。
<br>
<br>
<br>

## 概要
シンバイオントは、世界の最小単位である「素粒子」に直接干渉できる能力を持ちます。しかし、その強大な力と引き換えに「寿命の短縮」などの深刻な身体的負荷を負っています。タイプ分類能力の性質により、以下の3タイプ＋伝説上の1タイプに分類されます。
<br>
<br>

| Type | 名称 | 干渉対象（イメージ） |
| :--- | :--- | :--- |
| Type-L | Lepton(レプトン) | 電子などの軽粒子。電気や情報の制御。 |
| Type-Q | Quark(クォーク) | 複合粒子の構成。物質の結合や構造改変。 |
| Type-B | Boson(ボソン) | 相互作用の媒介。力（重力や衝撃など）の伝達。 |
| Type-T【伝説】 | Tachyon(タキオン) | 時間や因果への干渉を示唆する未知の存在。 |

*社会的希少性ウイルス感染からシンバイオントとして生き残るケースは極めてレアであり、世界に数えられるほどしか存在しません。ラプラススタンダードがヴェルナーガーデンを忌避・警戒する最大の理由は、この「制御不能な異能者」を生み出す環境にあります。*
