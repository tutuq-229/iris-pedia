Stitch Code（ステッチコード）

Stitch Code（ステッチコード）は、IRISの視覚情報インフラを支えるために設計された、低ノイズの長方形型コード規格である。人間には「青い幾何学模様」として美しく映り、スキャン端末にはきわめて高い精度で情報を提供する。つまりデザイン重視。

1.物理仕様

  • 解像度：96x32px
  • 基本配色：ブルー＆ホワイト



2.セクション構造

Stitch Codeは6個のレイヤーで構成される。

レイヤー名 配置 役割
リファレンス 右下 色のキャリブレーション。
縦パルスライン 右上 ヘッダー情報。
横パルスライン 左下 エラー訂正用ハッシュ。
ヴォイドライン 左と右を分ける。
ステッチ 左上 7本の隙間を持つバー。主要データを格納。
UV層 全域 紫外線反応層。追加のデータを格納。




3.エラー訂正と予測アルゴリズム

リファレンスキャリブレーション:

右下のリファレンスを照合し、照明環境や背景色による「色の変色」をデジタル補正する。

ギャップ・プレディクション:

上段7本のバーが一定のリズムで並んでいることを利用し、背景に埋もれた「本来あるべき隙間」を逆算して特定する。

ヴァリス・イェン符号:

下段データをパリティに割り当て、損壊データを数学的に復元する。
