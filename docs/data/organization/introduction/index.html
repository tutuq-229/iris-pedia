Nestism（ネスティズム）

Nestismは、伝統的な「神」を崇める団体ではなく、「世界の解明」を掲げる科学組織に近い宗教です。

トップ層：20代の若き7名が運営。

  • リーダー：サヴァン症候群的、ASD的な特性を持つ圧倒的天才。
  • 6人のサポート：各化学分野（量子力学、DNA、数学等）の専門家。リーダーの難解な思考を「教義」として翻訳・プレゼンし、SNSを通じて資金と支持を集めている。



活動内容

彼らは、この世界がシミュレーションのようなものと唱えており、存在しているであろう上位存在（神・設計者）の解析を掲げて、SNSで発信をしている。



FAQ

上位存在の干渉を証明する「バグ（不具合）」は見つかっているのですか？

量子力学の観測されるまで、量子はどこにでもあり、どこにもない（波の状態）というルールがゲームのレンダリングにそっくりだと、Nestismを信仰する者たちがよく言っています。またプランク長さという最小が存在することも、この世界の不具合であるといわれています。そしてさらに、伝説上のシンバイオントであるType-T（タキオン）の存在は、Nestismの間ではかなり濃厚なのではないかと言われており、これが存在するならばこの世界はシミュレーションだと言っています。タキオン粒子を扱えるものが存在するとしたら...という解説をよくNestism信者のコミュニティで見かけます。

彼らには裏の目的があるという噂を聞きました。

彼らは上位存在を提唱しつつ、下層を作っているのではないかと言われています。しかしこの情報はSNSなどでつぶやかれているだけで、本当のところはわかりません。

「Neptismに接触すること」は、一般市民や企業から見て、どの程度の「アウト」判定になりますか？

普通にSNSで活動しているくらいなので流行りとしてスルーされるでしょう。

彼らは「この世界」をサーバーにして、新しい世界をブート（起動）しようとしているのでは？

そこまで大きなプロジェクトを実行しようとしているかは分かりませんが、新たな実験場のようなものを作ろうとしているのではと言われています。この世界が光の速度をもとにした世界ならば、彼らが作っているのはタキオンの速度（虚数）をもとにした、反転した世界なのでは、みたいな噂があります。

彼らは「結果から原因を作る」力、つまりタキオンを求めている？

SNSで有名な考察では、例え下層でも、タキオンを操ることができれば、上層にも影響させることができるらしいです。つまりタキオンを操り、管理者権限のようなものを手に入れようとしているのではという話です。
