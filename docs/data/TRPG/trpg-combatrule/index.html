戦闘システム概要：APタクティクス

このTRPGは、限られたアクションポイント（AP）を駆使し、敵のシールドを崩して叩く、リソース管理型ターン性バトルです。


1.ターン構成とイニシアチブ

  • 先制判定：敵に「ばれているか・いないか」で先行・後攻が決定。
  • プレイヤーターン：味方サイドのターン内であれば、プレイヤー間の行動順は自由です。


2.アクションポイント（AP）

各プレイヤーは基本的に 3AP を保持してターンを開始します。 ターン中にできる行動は「攻撃」「防御」「特技」「その他」の４種類です。

攻撃

このTRPGでは、武器の性能（質）と投じたAP（量）の掛け合わせでダメージが決定します。

1．ダメージ計算の基本構造

ダメージは以下の数式で算出されます。 ダメージ = (消費AP) d (武器の威力値)

  • 武器の威力値：使用するダイスの面数を決定します。
  • 消費AP：振るダイスの個数を決定します。

2.アクション別ダイス配分表

アクション 消費AP ダイスロール入力例
小攻撃 1 1dx
中攻撃 2 2dx
大攻撃 3 3dx

※武器の威力をxとした場合の各アクションの挙動です。

防御

このTRPGでは、防具の防御力×APが防御力となります。

1．被ダメージ計算

被ダメージは以下の数式で算出されます。 被ダメージ = (敵の攻撃力) - (防御力)

  • 敵の攻撃力：敵の持っている武器や筋力に影響されます。
  • 防御力：自身の防御力×AP

2.防御配分表

アクション 消費AP 防御力
防御しない 0 0
小防御 1 1x防御力
中防御 2 2x防御力
大防御 3 3x防御力

例：攻撃に2AP・防御に1APを使うと、2dxダメージを相手に与えつつ、敵の攻撃ターンでは1x防御力分のダメージを防ぐ。

特技

特技は事前準備が必要なカスタマイズ要素です。使用するAP量は特技によって異なります。種類：持続の特技はAPを消費する必要はなく、常に能力が発動します。

  • セット制: あらかじめセット（装備）しておく必要があります。
  • 特技枠: 「ロール特技」と「通常特技」の2種類がありますが、共通の枠を奪い合います。枠の数はキャラクターの成長やロールによって増加します。
  • 成長要素: 使用するたびに特技に経験値が入り、特技自体のレベルが上昇・強化されます。
  • 連結特技: 特定の特技同士を組み合わせることで効果を変化させられます。

その他

アイテムの使用や逃げる、PCのアドリブなどがここに含まれます。

  • アイテムの使用： あらかじめ持ってきたアイテムを使用します。APは3消費します。
  • 逃げる： 1d2を振り、1が出たら、その戦闘から逃げることができます。要GM相談。
  • アドリブ：PCがとっさに思い付いた行動（石を拾って投げつけるなど）はこれに含まれます。消費APはGMと相談して決めてください。


3.シールド

特徴的な攻略要素です。

  • シールド値: 敵が展開している防御膜。
  • ダメージの法則: シールドが1点でも残っている限り、本体へのダメージは原則通りません。
  • ブレイク: まずは攻撃を集中させてシールド値を「0」にする必要があります。破壊して初めて、本体への直接ダメージが可能になります。


4.武器の消耗度

戦闘の代償として、武器は確実に摩耗していきます。

  • 減少タイミング: 戦闘終了時に消耗度が「1」低下。
  • 破損: 消耗度が限界に達すると武器は壊れ、使用不能になります。予備の武器を持つか、修理のタイミングを図る必要があります。
