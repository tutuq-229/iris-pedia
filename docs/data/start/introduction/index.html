このページは、環を持つ惑星「アイリス」の基本設定と、その世界を二分する勢力についての概要をまとめたものです。



🌏 惑星概要：アイリス (Iris)

アイリスは、空に巨大な環を持つ美しい惑星です。その環は鉱石「ロナザイト」の群によって構成されており、地表の生態系や技術発展に多大な影響を与えています。

-惑星名: アイリス (Iris)

-特徴: 巨大な惑星環を持つ。

-主要資源: ロナザイト鉱石


🏛 主要二大勢力

2. ヴェルナーガーデン (Werner Garden)
